# 🤱 NutriShield Mother Care
### *AI-Powered Maternal Nutrition & Anemia Alert System*

NutriShield Mother Care is a premium, state-of-the-art web application designed to empower expectant mothers by monitoring nutritional health and proactively identifying anemia risks. Built with a focus on preventive healthcare, it combines advanced AI analysis with a high-end, user-centric design to ensure a healthy pregnancy journey.

---

## ✨ Key Features

### 🔍 AI-Driven Diagnostics
- **Smart Symptom Checker**: Integrated camera scanning for pallor detection in eyes, tongue, and fingernails using simulated AI computer vision.
- **Blood Report Analyzer**: Automatic extraction and analysis of key parameters (Hb, RBC, Ferritin) from uploaded medical reports.
- **Anemia Risk Scoring**: Real-time risk assessment with severity classification and personalized health insights.

### 💊 Medication & Supplement Safety
- **AI Medication Scan**: Instant cross-referencing of consumed drugs against a database of substances to avoid during pregnancy (e.g., Warfarin, Isotretinoin, ACE inhibitors).
- **Safe Alternatives**: Suggests pregnancy-safe alternatives for common ailments after physician consultation.
- **Reminder System**: Integrated notifications for supplement intake (Iron, Folic Acid, etc.).

### 🥗 Personalized Nutrition
- **Customized Diet Plans**: Tailored meal schedules based on the user's specific anemia risk and nutritional needs.
- **Iron-Rich Food Guide**: Comprehensive database of nutrition-dense foods for maternal health.

### 📊 Maternal Health Tracking
- **Automated EDD Calculation**: Precision Expected Delivery Date tracking using Naegele's Rule.
- **Interactive Dashboard**: Visual health scores, pregnancy progress bars, and daily health tips.
- **Multilingual Support**: Available in **English**, **हिन्दी (Hindi)**, and **ಕನ್ನಡ (Kannada)**.

## 🚀 Getting Started

### Prerequisites
- Node.js (Latest LTS version recommended)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/nutrishield-mother-care.git
   ```
2. Navigate to the project directory:
   ```bash
   cd nutrishield-mother-care
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and navigate to the local URL (typically `http://localhost:5173`).
